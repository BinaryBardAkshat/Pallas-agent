# Video Script Writing

## Description
Write compelling video scripts for YouTube, social media, ads, or courses. Structure for maximum retention with strong hooks, clear flow, and effective CTAs.

## When to Use
- Writing YouTube video scripts
- Creating tutorial or explainer videos
- Short-form social media video content
- Product demos or ad scripts

## Instructions
**Video script structure**:

1. **Hook (0-5 seconds)** — The most important part. Must make viewers stay.
   - Ask a provocative question
   - State a surprising fact
   - Make a bold promise: "In 5 minutes, you'll know how to..."
   - Start mid-action

2. **Setup (5-30 seconds)** — Context and why they should care.
   - Establish the problem or opportunity
   - Connect to viewer's pain/desire

3. **Body** — Deliver on the promise. Use:
   - Numbered lists for clarity ("3 things you must know...")
   - Stories and examples over abstract concepts
   - Pattern interrupts every 60-90 seconds (question, statistic, demo)

4. **CTA (last 20 seconds)** — One clear action:
   - Subscribe / like for YouTube
   - Click link / buy / sign up for ads
   - Try it yourself for tutorials

**Formatting for scripts**:
- [VISUAL]: what should be shown on screen
- [PAUSE]: natural pause point
- Write how people SPEAK, not how they write — contractions, short sentences

## Examples
- "Write a YouTube script explaining how neural networks work"
- "Script a 60-second product demo video"
- "Write a hook for a video about productivity hacks"
- "Create a tutorial script for this Python library"
